# Contributing

Thanks for contributing.

## What you can contribute
- fix factual errors
- improve wording
- add missing citations
- suggest new interview questions
- expand incomplete answers
- flag outdated material

## Style guide
- keep answers concise but technically accurate
- prefer primary sources when possible
- avoid hype and vague claims
- explain trade-offs clearly
- write for AI engineer interviews, not research papers

## Format
For each question, use:

### Question
**Answer**
**Sources**

## Before submitting
- check for duplicates
- keep terminology consistent
- make sure links work
